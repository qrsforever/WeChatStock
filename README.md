# WeChatStock

WeChatStock: talk about stock with wechat


# Required Dependences

- `sudo pip install -U wxpy` [-i "https://pypi.doubanio.com/simple/"]

- `sudo pip install -U pillow` 终端显示二维码

- `sudo apt-get install okular` 看epub文档
